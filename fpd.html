Фонд пространственных данных

Основной целью проекта «Создание и развитие базовых информационных ресурсов» является обеспечение информационной совместимости прикладных ведомственных, а также иных информационных систем в составе информационного общества Ленинградской области.

В рамках реализации проекта были проведены следующие мероприятия:

  • Внедрение инфраструктуры пространственных данных в практику государственного управления регионом. В рамках данного мероприятия выполнены следующие работы:
    • Cоздан интерфейс для отображения, учета и контроля работы предприятий торговли, общественного питания и оказания бытовых услуг населению и обеспечения подготовки различных отчетных материалов для руководства и вышестоящих организаций, для планирования работы по развитию потребительского рынка
    • Cоздан интерфейс для учета правонарушений несовершеннолетних, ведения дел правонарушений несовершеннолетних, и иных действий по правонарушениям, а также подготовке различных отчетных и справочных материалов
    • Cоздан интерфейс Центра обработки обращений населения для автоматизации деятельности многоуровневых служб, осуществления информационного взаимодействия между ними
    • Cоздан интерфейс для информационного обеспечения деятельности по автоматизации деятельности органов исполнительной власти Ленинградской области, занимающихся распределением путёвок в детские и молодёжные оздоровительные лагеря
    • Cоздан аналитический интерфейс для комплексной аналитики разнородных данных по управлению областным хозяйством
    • Cоздан интерфейс удаленного доступа к инфраструктуре пространственных данных ЛО
    • Cоздан интерфейс для предоставления в рамках мобильного терминала информации, необходимой для принятия управленческих решений
  • Наполнение баз данных базовых информационных ресурсов следующими пространственными данными:
    • Актуальные цифровые топографические карты открытого пользования на территорию Ленинградской области в системе координат МСК 47 и WGS 84 масштабов: 1:1 000 000, 1:500 000, 1:200 000, 1:100 000, 1:50 000, 1:25 000
    • Выполнено уточнение навигационных графов и адресных планов на населенные пункты Гатчинского муниципального района, Всеволожского муниципального района Ленинградской области и уточнение навигационных графов на межселенной территории масштабов: 1:50 000 , 1:25000
